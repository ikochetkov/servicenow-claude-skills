# ServiceNow Official Figma Library Catalog

Complete inventory of ServiceNow's official Figma design libraries published on Figma Community.
Part of the **Horizon Design System**, updated with each major ServiceNow release.

**Figma Community Profile:** https://www.figma.com/@servicenow
**Horizon Design System:** https://horizon.servicenow.com/
**Archived Releases:** https://github.com/ServiceNow/FigmaResources

## How to Use These Libraries

1. Go to the Figma Community URL for the library
2. Click "Open in Figma" to duplicate to your account
3. Enable it as a library in your project for component access
4. Components include properties, variants, and nested instances matching UI Builder (UIB) behavior
5. Use Theme Builder in ServiceNow to customize, then update Figma variables to match

## Release Cadence

Figma libraries update for each major ServiceNow family release (twice yearly, named after cities). Updates publish ~30 days after GA.

Recent releases: Utah → Vancouver → Washington DC → Xanadu → Yokohama → Zurich (current).
