# App Engine (Creator Workflows) Pricing & Packaging

## Overview
App Engine is ServiceNow's low-code application development platform for building custom workflows and applications. Licensed by **Fulfiller**, **Unrestricted User**, or **Application User**.

---

## Pricing Models

### Fulfiller Model
Role-based: Users fulfilling requests, doing work, or managing processes.

### Unrestricted User Model
Every active user in sys_user table with full rights subscription.

### Application User Model
Users who can access one or more custom applications within the purchased product.

**Note:** App User count includes requestors + potential external users accessing custom applications.
